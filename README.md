# Theme Hook App - useColorScheme 

A React Native demo exploring dynamic theme switching using the `useColorScheme` hook.

##  Features

- **Auto Theme Detection**: Automatically switches between Light and Dark modes.
- **Dynamic Palette**: Uses a central palette for background, text, and accent colors.
- **Visual Debug**: Displays the current `colorScheme` state for verification.

##  Getting Started

1. Navigate to the project directory:
   ```bash
   cd theme-hook-app
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Expo server with tunneling:
   ```bash
   npx expo start --tunnel
   ```

##  Structure

- `src/screens/WelcomeScreen.js`: Theme logic and UI.
- `assets/logo.png`: Bistro logo.
- `App.js`: Root component.
